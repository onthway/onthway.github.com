资源

VIM教程与学习资料汇总

使用everything进行文件搜索，对findeverything.vim进行修改，不进行目录输入，代码修改如下:

    "let dir = input("Find in dir: ", "", "dir")
    "let dir = s:Handle_String(dir)
    "let cmd = cmd .' '. g:fe_es_option . ' ' . dir . ' ' . pattern
    ==>        
    let cmd = cmd .' '. g:fe_es_option . ' ' . pattern

插件

  1. vundle 管理插件的插件

  2. SuperTab 使用tab键进行补齐操作，默认为上一次补齐方式

    vim中其它的补全方式包括：

    整行补全                        CTRL-X CTRL-L
根据当前文件里关键字补全        CTRL-X CTRL-N
根据字典补全                    CTRL-X CTRL-K
根据同义词字典补全              CTRL-X CTRL-T
根据头文件内关键字补全          CTRL-X CTRL-I
根据标签补全                    CTRL-X CTRL-]
补全文件名                      CTRL-X CTRL-F
补全宏定义                      CTRL-X CTRL-D
补全vim命令                     CTRL-X CTRL-V
用户自定义补全方式              CTRL-X CTRL-U
拼写建议                        CTRL-X CTRL-S

  3. Markdown markdown语法插件

    markdown插件有多个版本，有几个版本的插件无法正常高亮使用，原因不详，该链接插件可以正常工作
    该插件需要放在$VIMRUNTIME下才可以正常运行，原因不详

    tips

  4. 使用vundle进行插件管理

    运行以下命令：

    cd vim\vim73\
mkdir bundle
cd  bundle
mkdir vundle
cd vundle
git clone http://github.com/gmarik/vundle.git
  5. 使用supertab插件后可以通过tab键进行补全输入，关键是支持中文，感觉不错
  6. 重新安装后vundle管理的插件需要重新安装,vundle需要依赖git命令,需要在path中加入git命令所在的目录,如果发现加入后没有效果,可以重启计算机使设置生效.
