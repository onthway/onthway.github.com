vim /private/var/mobile/Library/Preferences/XBMC/userdata/Advancedsettings.xml

<network>
<cachemembuffersize>0</cachemembuffersize>

<gui>    
<algorithmdirtyregions>1</algorithmdirtyregions>

<tvshowmatching append="no">
<regexp>\[[Ss]([0-9]+)\]_\[[Ee]([0-9]+)\]?([^\\/]*)(?:(?:[\\/]video_ts)[\\/]video_ts.ifo)?</regexp>
<regexp>[\._ \[\-\\/]([0-9]+)x([0-9]+)([^\\/]*)(?:(?:[\\/]video_ts)[\\/]video_ts.ifo)?</regexp>
<regexp>[Ss]([0-9]+)[\.\-]?[Ee]([0-9]+)([^\\/]*)(?:(?:[\\/]video_ts)[\\/]video_ts.ifo)?</regexp>
<regexp>[\._ \-\\/]([0-9]+)([0-9][0-9])([\._ \-][^\\/]*)(?:(?:[\\/]video_ts)[\\/]video_ts.ifo)?</regexp>
</tvshowmatching>
<latency>
<delay>0</delay>
</latency>
</advancedsettings>

Then place this file in the following directory (depending on your platform).

Windows XP: Documents and Settings\<your_user_name>\Application Data\XBMC\userdata\
Vista/Windows 7: Users\<your_user_name>\AppData\Roaming\XBMC\userdata\
Mac OS X: /Users/<your_user_name>/Library/Application Support/XBMC/userdata/
iOS: /private/var/mobile/Library/Preferences/XBMC/userdata/
Linux: $HOME/.xbmc/userdata/

如何支持lotus notes

如果lotus notes重新安装或者采用绿色方式安装lotus notes，此时googledesktop将不能对lotus notes的邮件进行索引，解决办法如下：

  • 检查notesgdsplugin.dll是否在notes目录，如果不在，说明没有正常部署

  • 使用7-zip解压GoogleDesktopSetup.msi可以提取出notesgdsplugin.dll.5DD2E499_D46C_4A16_B6E2_42E3B3376A3F 文件，将后缀删除，拷贝到notes目录

  • 在notes.ini文件中增加AddinMenus=notesgdsplugin.dll

  • 运行notes，根据提示进行设置，运气不坏，即可搞定

樊正伦老师的五运六气判断

根据《黄帝内经》推算，从2012年1月下旬至2013年1月下旬，是为壬辰年的气运周期。壬辰年为木运太过之年，太阳寒水司天，太阴湿土在泉。大体是上半年偏寒，下半年多雨，全年气候以寒湿为主，而且气运会先时而至，比正常的节气变化来得早些。

　　初之气：从大寒至春分前（1月中下旬至3月中下旬），主气是厥阴风木，客气是少阳相火。“气乃大温，草乃早荣”，延续了上年的暖冬，是一个暖和的初春，“冬不藏精，春必病温”，人们易感疫病，是瘟病多发之时。要注意提早预防，清内热，防上火，多吃梨、藕、橙、萝卜、荸荠，也可以用冰糖、白茅根、芦根等煮水喝。

　　二之气：从春分至小满前（3月中下旬至5月中下旬），主气是少阴君火，客气是阳明燥金。“大凉反至”，恰逢停止供暖，导致热郁于内而难发散，气候反寒，可能多发风寒感冒咳嗽。要注意“春捂”，不要急于减衣服。春行秋令，干燥凉爽，除了多吃上述清内火的食品，更要多喝水，多到室外活动。

　　三之气：从小满至大暑前（5月中下旬至7月中下旬），主气是少阳相火，客气是太阳寒水。正是司天之气起主要影响的时期，天气该热而不热，寒气笼罩，开始降雨，易因内郁热外受寒而得病，由于寒气侵袭，湿气渐起，可适当吃些温寒燥湿的食品，比如小米、红小豆、薏米等。

　　四之气：从大暑至秋分前（7月中下旬至9月中下旬），主气是太阴湿土，客气厥阴风木，“风湿交争，风化为雨”，应是丰收年景。但由于湿气困脾，脾主四肢肌肉，应注意防风湿，应多吃健脾或苦味食品，比如山药、薏米等。

　　五之气：从秋分至小雪前（9月中下旬至11月中下旬），主气是阳明燥金，客气是少阴君火，“阳复化，草乃长……民乃舒”，这可能是一年中最舒服、最少疾病的时期。

　　六之气：从小雪至大寒前（11月中下旬至2013年1月中下旬），主气是太阳寒水，客气是太阴湿土。“湿令行，阴凝太虚，埃昏郊野……寒风以至”，是一个又湿又冷、风大扬沙的冬季，可以多吃温热性的食品，比如牛羊肉等，这个时期也是进补的大好时机。

云中阁主对2012初之气的分析和预测

2012年初之氣（1月21至3月19）：主氣為厥陰風木，客氣為少陽相火加臨。

呈風寒濕熱之稟氣，此時，由於主客氣形成風火相煽，便“寒氣早化，氣乃大濕，地氣遷，草用榮”的一派風溫、濕溫氣象，導致“民乃厲，溫病乃作，身熱頭痛，嘔吐，肌腠瘡瘍”出現，現代醫學的流行性感冒、手足口病、肝炎、腸胃炎、腦炎，及風疹、瘡疹、痛風、中風等疾患，甚者危及生命，故應注意滋養腎水，涵肝木、健脾燥濕清熱解毒調理，以保健康。
